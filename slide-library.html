About

One-line positioning statement. The sentence that makes the direction obvious.

Reference mindset: one cultural anchor × one material anchor → reimagined for the product. Two sentences, no more.

A System of —
  • Rule one — what leads (photography / type / color).
  • Rule two — the single accent and how it's used.
  • Rule three — the signature move that recurs.
  • Rule four — what's deliberately kept imperfect.
Visual Recommendations
  • Canvas — the default surface, not pure white.
  • Type — family, weight, and the size floor.
  • Imagery — one per surface; no collages in-product.
  • Motion — speed and character in one phrase.

An argument that earns three to five sentences of support.

Medium is where an idea that needs a paragraph lives — too long for a headline, too important for a footnote. Two short paragraphs, never more. Leading matches the body scale elsewhere in the deck so the slide feels of a piece. The surrounding whitespace is the design.

02 · Statements

Three beliefs, side by side.

01 · Form

Design that reads as design, not decoration.

Every element earns its place. No trope imports, no slop gradients. When we use color we use one, and we use it sharp.

02 · Work

Slow answers beat fast ones we'd regret.

We do fewer things so each one lasts. The brief is to be remembered in a year, not recalled by the algorithm in a week.

03 · People

Warm by default. Strict when it matters.

Kindness is the first instinct. Standards are the second. Both are load-bearing; neither is optional.

04 · Lists & Tables

How we work.

Numbered · Long-form
01

Start with the argument.

A week of listening before a pixel moves. We leave with the one sentence the project is trying to prove.

02

Make it small, then make it right.

One surface — a cover, a home, a stylescape — stress-tested before the system gets built around it.

03

Write it like we'll teach it.

Rules live in the file. Anyone who opens the library a year later can pick it up without the designer in the room.

04

Ship the system, not the screens.

The deliverable is the library — linked, documented, and ready for a tenth surface we haven't drawn yet.
